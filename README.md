# TWU Academic Advisor Navigator

A web app that helps Trinity Western University (TWU) students organize key academic and student-life information in one place. Think of it as a lightweight student “planner” for staying on track with scheduling, graduation progress, and campus resources.

## Live App
No installation required — open in your browser:  
[https://merrill.in/academic-advisor/index.html](https://merrill.in/academic-advisor/index.html)

## Key Features

### Course Information + Reviews
- View course information (e.g., CMPT 113 / CMPT 123).
- Read reviews submitted by other students.
- Submit your own review.

### Course Schedule (Timetable)
- Add courses by entering a course name, choosing a time slot, and selecting days
- Courses persist in the timetable until reset
- Schedule data is stored so it remains when you return.

### Student Life
**Housing**
- Post housing listings (title, price, address, location, name)
- Browse listings posted by other students.

**Writing Centre**
- Centralized access to workshops, WriteAway, resources, appointments, and tutoring/coaching options.

### Campus Maps
- Campus building map experience implemented using the **Google API**.  

### Credit Progress (Major Checklist)
- Major checklist with **4 sections**
- Add course entries with: course name, description, semester hours
- Automatically calculates semester hours per section + overall total
- Reset available per section for safety/accuracy
